答案与核对

答案已校验

3

页面按“逐图判断”的方式收口：5 个图里最终保留 3 个，另外 2 个因为折起后会出现重叠或对面关系不成立而被排除。

推荐思路

先把每个图都找一个“中心格”

不要五个图一起想。每看一个图，先找最适合作为中心的那一格，再看周围面是往哪几边展开。

优先排除会撞面的图

如果某个末端翻起来后会和另一面压到同一位置，这个图就直接排除，不用继续往下想。

最后只统计保留下来的图

逐图判断后，把标成“保留”的图数出来即可。本题页面按终审口径收成 3 个。

5个图逐个判断

图1保留

图1：保留

中间竖着的两格可以带起左右和下方各一面，6 个面折起后能分到立方体的 6 个不同方向，不会重叠。

复盘提示：这一图可以先把中间那格看成正面，再让右边和下边依次翻起来。

图2保留

图2：保留

虽然右侧多出一个“拐下去”的尾巴，但折起后它正好成为最后一个独立面，不会和前面任何一面撞位。

复盘提示：先看中间横着的 3 格，再把最右下那格想成最后盖上的一面。

图3排除

图3：排除

右端往下拐出的那一格折起来后，会和左上方向的一面争同一个位置，出现“撞面”，所以不能折成正方体。

复盘提示：这类“长条末端再往下拐”的图，最适合检查最后一格会不会压回到前面已经占掉的位置。

图4保留

图4：保留

上面 3 格和下面 3 格是错位连接的，折起后上下两排会落到不同方向，6 个面仍然能完整分开。

复盘提示：不要因为它看起来“很长”就先判错，关键是看折起后每一格去到的方向有没有重复。

图5排除

图5：排除

这一个在原题截图里位于最右侧，页面保留了终审结论：折叠后会出现方向重合，不能作为正方体展开图。当前站内资源只保留到边缘裁切图，因此这里用结论型说明收口。

复盘提示：做真题时，如果某个图已经被终审标记为排除，复盘重点就放在“为什么会方向重合”，而不是重复硬记答案。

常见卡点

  • 把 5 个图一起在脑中转，结果中途混掉，不知道刚才判断到了哪一个。
  • 只看平面上能不能连起来，没有检查折起后会不会压到同一面。
  • 看到长条形就先入为主地判错，没有逐图去找中心格和相邻关系。
