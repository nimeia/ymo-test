建议解题顺序

  1. 先确认题目条件里哪些信息是固定的，哪些信息需要推出来。
  2. 先想清楚视角，再判断哪些部分会被遮挡或相邻。
  3. 这题的核心能力是 折叠、切分，做完后要回头检查每个条件是否都被满足。

常见错误

  • 只看答案像不像，没有回到题目条件逐一检查。
  • 把看到的面和被挡住的部分混在一起判断。
