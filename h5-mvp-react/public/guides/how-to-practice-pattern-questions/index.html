先看什么

先观察数量、位置、方向、颜色、大小这几个常见维度，不要一开始就盯着结果猜。

如果一条线看不出规律，可以试着横向看、纵向看，或把奇数位和偶数位分开看。

怎么表达规律

训练时要鼓励孩子把规律说出来，比如“每次加 2”“黑白交替”“左右轮换”。

一旦能说清楚，很多题就会从“感觉像”变成“我知道为什么”。

怎么检查答案

选出答案后，最好再倒回去检验一次，看它是否和前面的每一步都一致。

对于复杂题，可以先排除明显不符合规律的选项，再在剩余选项里比较。
